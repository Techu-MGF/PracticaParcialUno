¿Qué es DNS?

Es un protocolo que traduce los nombres de dominios aptos para la lectura humana a direcciones IP para la lectura por parte de las máquinas

¿Qué es IP y el puerto 8080?

El Protocolo de Internet es un conjunto de reglas que rigen el formato de los datos enviados a través de Internet. Las direcciones IP son el identificador que permite el envío de información entre dispositivos en una red.

Por otro lado, el puerto 8080 es aquel que se ejecutra en localhost, sirve como entorno de prueba para desarrolladores y se utiliza para depurar y probar aplicaciones web
